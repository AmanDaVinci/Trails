# Trails
A tool to save, categorize and share entire web sessions with a visual map of the surfing flow.

Modules:

#1) Extension (Ayush and Swapnil)
  1.1) Save single link 
  1.2) Save Session
    1.2.1) Add name / tags

#2) Website
  2.1) Search Bar
    2.1.1) Tag
    2.1.2) Session Title
  2.2) Website Thumbnail
    2.2.1) Screenshot
    2.2.2) Title & Tags
  2.3) Flow Stack
    2.3.1) Map Dynamic Page
    2.3.2) Share
    
#3) Trails Engine
  3.1) Database(NoSql) (Aman and Subhojeet)
    3.1.1) Link Address
    3.1.2) Tags
    3.1.3) Screenshot
    3.1.4) Time
  3.2) Tag System
  3.3) Map
    3.3.1) Time
    3.3.2) Transition type
